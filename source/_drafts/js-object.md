# JS 对象分类

- 宿主对象(host Object)：如浏览器等
- 内置对象(Built-in Object)：JS 语言提供
  - 固有对象(Intrinsic Object) ：由标准规定，随 JS 运行而自动创建对象实例
  - **原生对象(Native Object)**：可以由用户通过 Array、RegExp 等内置构造器或者特殊语法创建的对象。
  - 普通对象(Ordinary Object)：用户自己定义
